📖 使用说明

▼

📌 基本操作

  • 点击日历中任意日期的 + 号可快速创建新排期
  • 点击日历上的排期横条可查看详情并编辑
  • 排期支持设置开始和结束日期，以横条形式跨日展示
  • 单日排期显示为圆角标签，多日排期显示为贯穿横条
  • 拖拽排期横条到其他日期可快速调整时间，持续天数不变

📅 日历标记

  • 红色文字：显示中国传统节日与公历节日名称
  • 粉色背景 + 绿色「休」：法定节假日放假日
  • 黄色背景 + 橙色「班」：调休补班工作日
  • 已内置 2025-2027 年法定节假日及调休安排

🔍 筛选与导航

  • 使用顶部筛选栏可按状态快速过滤排期
  • 支持五种状态：待发布、已发布、待审核、待结算、已结算
  • 点击「今天」按钮快速回到当前月份
  • 使用左右箭头切换查看不同月份

🔥 平台热点

  • 页面底部实时聚合展示抖音和微博热搜榜
  • 支持按分类筛选热搜（二次元、娱乐、影视综、游戏等）
  • 分类匹配不足时自动关联相近分类扩展结果
  • 搜索框可输入自定义关键词实时过滤
  • 点击热搜条目可跳转到对应话题详情页
  • 数据自动缓存 30 分钟，点击「刷新」获取最新

📤 数据管理

  • 导出：将所有数据导出为 JSON 文件，可用于备份或分享
  • 导入：从 JSON 文件导入排期数据，会与现有数据合并
  • 所有数据自动保存在浏览器本地，刷新不丢失

⏰ 提醒与快捷键

  • 创建排期时可设置提前提醒（5分钟~1天）
  • 页面打开时自动检测并弹出浏览器通知提醒
  • 按 Esc 快速关闭任意弹窗
  • 点击弹窗外部半透明区域也可关闭弹窗
